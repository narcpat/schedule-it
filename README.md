# Schedule-it

## Purpose

To create a workday scheduler which displays today's date, the different hours of the work day, and any saved tasks for the day. The save tasks must be retrievable when the page is later reloaded.

## Built With

- HTML
- CSS
- Bootstrap
- JavaScript
- jQuery
- Moment.js
